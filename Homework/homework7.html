Homework 7 - GoldField: Search for Mover

This is the first homework that will use the goldfield API. If you have any questions, you can read more about the API at that link.

  1. Create a web page and put a tag on the page for GoldField to use. Create the GoldField in code and fill it with a random number of obstacles.
  2. Place the hero and one monster on the field. These should be placed at random.
  3. Use a pair of for loops to LookAt each block in the field. When you find the hero record the position. When you find the monster, record the position.
  4. As soon as you have found both of them figure out the Δx and Δy between them. Create an output tag and write out something like this:
    The Monster is 4 right and 3 up from the Hero.
  5. Create a reset button that clears the field and does it all again.
  6. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should be able to check your code again and again to see if it gives you the right answer.
  7. Submit the link to the page in Canvas.
