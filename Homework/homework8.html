Homework 8 - GoldField: Look-plan algorithms

This is the first homework that will use the goldfield API. If you have any questions, you can read more about the API at that link. This homework will create a very stupid AI--something like the ghosts in Pac-Man.

  1. Start with your homework 7. You will need the same steps in your response to this homework, so make a new copy.
  2. When the field is created, save the Monster and Hero to variables.
  3. Set up a heartbeat and when the beat occurs, call a move function.
  4. In the move function,
    1. Figure out the Δx and Δy from the Monster to the Human.
    2. Pick the best direction to move to get closer. For example, you might always go left/right first and then up/down.
    3. Figure out the new x,y location where you want to move.
    4. LookAt that location. If it is empty, SetPosition for that spot.
  5. See what happens when the field is empty of obstacles. What if it is very full? Is this a bad thing in a game? Is this a thing a human player would do?
