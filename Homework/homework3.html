Homework 3 - Locate items on GUI

  1. Create a simple game where the computer asks you to click on the right place.
  2. Create a canvas tag and slect a random x and y coordinate on the screen.
  3. Ask the user to click on that place. Capture the x and y of the mouse event location.
  4. Score the user based on how close the click was to the intended target. Maybe show where the right location was.
  5. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should be able to play the game by yourself to test your GUI location skills.
  6. Submit the link to the page in Canvas.
