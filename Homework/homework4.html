Homework 4 - Create a clock

  1. Put three labels on the screen for hours, minutes, and seconds.
  2. Create a 500 ms time event.
  3. When the time event happens, work out the current hour, minute, and second. Update the labels.
  4. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should be ableto play the game by yourself to test your GUI location skills.
  5. Submit the link to the page in Canvas.
