Lab 2 - Father-son "catch"

Apply the lessons about fields, functions, and events. Create two buttons and p tags. Give the tags the ids father and son. Create add an onClick event for each button. In each case call a function with the id of the other person. When the function is called change the text (innerHTML) of the other label to read "ball".

When you push each button it should "throw" the ball to the other player. Put all this into a single web page. Submit a link to the lab via canvas.
