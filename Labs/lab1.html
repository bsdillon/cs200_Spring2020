Lab 1 - Experimenting with style

Apply the lessons from lecture #1 by creating a small web page and applying styles the various elements using tag, class, name, and ID. Try each of the following:

  • Create two p tags. Give one a class. Define styles for each one.
  • Create a new p tag with a name. Create a h1 tag with the same name. Define a style for that name.
  • Create a final p tag with an ID. Create a style for that ID.
  • Try making changes to the style for the whole body. Which tags are affected?
  • In each of the tags, put text describing how it behaves and why.

Put all this into a single web page. Submit a link to the lab via canvas.
