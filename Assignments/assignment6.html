Assignment #6 - Inheritance vs Composition

Purpose
Determine the difference between and advantages of inheritance and composition.
Instructions
  1. Define an object for "Car". The car object must have members for color, speed, and name. Make the name "Car"
  2. Create a function that will write text out to the screen. When you create a new car, have it print its name and color as in I am a green Car and I go 10 mph.
  3. Use inheritance to create Car child types. Let each type define its own name and speed.
  4. Test the effect of creating several child type instances. It should print out something like this:
    I am a green Car and I go 10 mph.
    I am a red Mustang and I go 100 mph.
    I am a blue T-Bird and I go 50 mph.
    I am a silver Lamborghini and I go 200 mph.
  5. Create a new object for "Animal". This animal object must have members for color and fact. Make name a parameter. Make the animal print out using the same function BUT let it call print functions for each of the color and fact members.
    animal.color.print(); animal.fact.print();
  6. Create an object Red that prints "red". Create a fact Flier that prints "I can fly".
  7. Create an instance of the Animal giving it the parameters "Bird", a Red object, and a Flier object. It should print out
    I am a red Bird and I can fly.
  8. Up until now, you have created two very different sets of objects, one using inherited behavior and one using composed behavior. Now we will see why some people believe composition is always better.
  9. Create an instance of each of your cars. Create an instance of the Bird. Print out a line saying "These are the original types".
  10. Use your family of car objects that print the following. Count how many lines of code it takes.
    I am a blue sedan and I go 20 mph
    I am a blue sedan and I go 30 mph
    I am a red sedan and I go 20 mph
    I am a gold minivan and I go 20 mph
    I am a blue minivan and I go 20 mph
    I am a red minivan and I go 20 mph
    I am a gold minivan and I go 30 mph
  11. Use your Animal object to print the following. You may need to create new objects like Red and Flyer. Count how many lines of code it takes.
    I am a blue snake and I slither
    I am a blue snake and I fly
    I am a red snake and I slither
    I am a green parrot and I fly
    I am a blue parrot and I fly
    I am a red parrot and I fly
    I am a green parrot and slither
  12. Print out each group of objects followed by the number of lines you added.
  13. Add at least one paragraph explaining the results. Which method takes more work? Why?
  14. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs.
  15. Submit the link to the page in Canvas.
