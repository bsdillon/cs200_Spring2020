Assignment #2 - Active style change

Purpose
Allow students to define several style guidelines in CSS and switch between these guidelines using a simple button.
Instructions
  1. Create a web page with at least three paragraphs of text from a favorite book or poem.
  2. Create at least two sets of CSS instructions based on competing class definitions for the body tag. You may use the examples below to investigate, but you will not receive credit unless you create your own CSS instructions.
    body.nox {color:white; background-color:black;}
    body.lumos {color:black; background-color:white;}
  3. Create buttons corresponding to each of the CSS instructions. This code says that when someone clicks on the button, it will change the className to 'nox'.
    <button onclick="document.body.className='nox'">Nox</button>
  4. You might read these examples.
  5. Put all these elements in a page. Test to make sure the buttons work and resolve any bugs. When you push a button, the style of the page should change as with the buttons below.
  6. Submit the link to the page in Canvas.
