Assignment #3 - Multiple catch games

Purpose
Extend the work from lab 2 using parameters to change the behavior with a single method.
Instructions
  1. Put at least five of <p> tags and buttons in random places on the screen using the techniques from lab #2.
  2. Create a throw function with a parameter sender.
  3. Each tag should have a unique id with a number i.e. id="tag1".
  4. Each button should have an onclick event that calls the throw function with the corresponding number as a parameter e.g. onclick="throw(1)".
  5. Write the throw function similar to that used in lab 2. Instead of directly calling a known tag, you can select a random tag.
    • You will need to use the random function to generate a number in the range of your buttons/labels.
    • You should check if the random number is the same as the sender. We don't want to throw back to ourselves.
    • If the numbers match, throw again.
    • If the numbers don't match, throw to the new tag using the random number.
      var catcher = document.getElementById("tag"+number);
  6. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. The ball should move randomly around the screen.
  7. Submit the link to the page in Canvas.
