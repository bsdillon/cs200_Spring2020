Assignment #7 - GoldField: Buttons and mouse events

Purpose
Use the GoldField API to connect keyboard and mouse events.
Instructions
  1. Create a function for keyboard events and another one for mouse events.
  2. Create the GoldField and use SetClickListener and SetKeyListener to connect the two functions.
  3. In the key function, check if the event number (i.e. event[0]) matches one of these numbers. Create the indicated Mover when that key is pressed.
    • 97 (a) Monster
    • 115 (s) Gold
    • 100 (d) Tower
    • 102 (f) Water
    • In the mouse function, check if there is an object at that location (i.e. goldField.GetMover(contents[0]);) and call it's Destroy method.
    • Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should be able to create an object with a key press and delete it with a mouse click.
    • Submit the link to the page in Canvas.
