Assignment #9 - GoldField: Mover chase scenes

Purpose
Simulate intelligent "following" behavior in the GoldField API
Instructions
  1. Create GoldField and add a Hero and a Monster.
  2. Add controls so you can move the Hero.
  3. Add a heartbeat to the program. When the clock ticks every second, the Monster will move.
    • Calculate the distance x and y between the monster and the Hero. These values have direction so -x is to the left and +x is to the right.
    • Decide which direction you need to move to get closer to the Hero. How you do this will affect the behavior of the Monster.
    • Move the Monster one step in the rigth direction.
  4. Add a block of gold. When the clock ticks, it should also move AWAY from the human. Use a similar method to that used with the monster. Remember to check for collisions as the gold WILL run into walls.
  5. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should have a minigame where the user can chase the gold while the monster chases the human.
  6. Submit the link to the page in Canvas.
