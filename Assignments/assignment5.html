Assignment #5 - Heartbeat and motion

Purpose
Use time and GUI functions to create movement metered by a heartbeat function.
Instructions
  1. Create a clock function that updates every 250 milliseconds.
  2. Define an object that will bounce around the screen. It doesn't have to be a ball.
  3. Create an array of objects.
  4. When the clock "ticks" make each object in the array move.
  5. Create a button and function that creates one new instance of the object and adds it to the array. You may consider buttons that also add 10 or even 100.
  6. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. I suggest testing with ONE object before trying many more.
  7. Submit the link to the page in Canvas.
