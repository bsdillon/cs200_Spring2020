Assignment #8 - GoldField: Collision checking

Purpose
Implement game logic that allows Movers to interact.
Instructions
  1. Create GoldField and add a hero with a mix of several gold and monsters.
  2. Implement keyboard movement so you can control the hero.
  3. Put <p> tags for coins and lives on the page.
  4. Create a function that will take care of destroying the hero, count the lives lost, write them to the page, and create a new Hero randomly.
  5. Create a function that will take care of destroying the coin, count the coins found, and write them to the page
  6. When the hero moves, look at the new space first.
    • If the space is empty, move the hero into the space.
    • If the space has a monster, destroy the monster and call the function to destroy the hero.
    • If the space has a coin, call the function to destroy the coin, then move the hero into the space.
  7. Put all these elements in a page. Test to make sure the buttons work as expected and resolve any bugs. You should have a little mini-game where the hero collects coins and dies if he hits a monster.
  8. Submit the link to the page in Canvas.
