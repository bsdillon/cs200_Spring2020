The Hero is an object that has its own variables for x and y position. Within the Hero object, the keyword this just means 'this object'. So the Hero's location is always known as this.xPos and this.yPos. The Hero always knows where it is using these variables. We can write out the position as a pair of x and y values. In computers, this is called a tuple. This is the Hero's current position.


(this.xPos, this.yPos) = (,)
